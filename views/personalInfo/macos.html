    黑苹果的安装

    蚝仔烙 10/23/2020 黑苹果

    macos被黑客破解之后可以安装在PC上，从而出现了一大批未购买苹果机而使用苹果操作系统的机器，这种就被称为黑苹果(Hackintosh)，在mac苹果机上面安装原版mac系统的被称为白苹果

    三星PM981(a) / PM991和美光2200S NVMe SSD 这些固态硬盘不兼容（导致内核崩溃），因此需要NVMeFix.kext来修复这些内核崩溃。请注意，即使使用NVMeFix.kext，这些驱动器仍可能会导致启动问题。

    本教程参考黑果小兵实操完成

    软件准备 md5检查器：WinMD5 U盘制作工具：transmac 需要安装的镜像

    链接: https://pan.baidu.com/s/1qznZCqI85CkM2Asib9WtSg 密码: 6f0o

    # 一、检验镜像的md5值

    利用刚才下载的WinMD5检查md5值是否正确，如果md5值不相同必须重新下载安装镜像(懒得检测就别检，装不了再换)

    # 二、将安装镜像写到USB上 16G及以上（制作安装镜像）

    下载balenaEtcher，选择安装镜像，选择需要制作的U盘，点击 Flash 即可。Windows10需要以管理员权限运行 制作完成后自动帮你弄了EFI分区了，里面集成了常用硬件的EFI文件 （如果USB安装盘自带的EFI无法完成安装或者安装后不完美，那么就需要执行替换EFI的操作） 查找适合自己的EFI可以到这些网站 远景：http://bbs.pcbeta.com tonymacx86: https://www.tonymacx86.com 操作过程暂略

    # 三、设置BIOS

    以联想天逸510s Mini为例： 安全菜单： 安全启动 -> 关闭 (Disable Secure Boot) 高级菜单： CFG Lock -> 关闭 (Disabling CFG Lock) 设备： 配置SATA为 -> AHCI 预指派内存大小：64MB (DVMT pre-allocated memory) 显示设备 ATA设备菜单： 其它参数默认即可

    # 四、安装macOS Big Sur

    开机，按F12选择U盘引导，光标移动到EFI USB Device选择OpenCore分区启动： 进入OpenCore主引导界面，选择Install macOS Big Sur，直接回车进入OpenCore引导，这期间会显示引导日志

    这个过程需要1-2分钟,耐心等待，进入安装程序,出现语言选择界面，选择中文

    出现安装界面，选择磁盘工具，点击继续

    进入磁盘工具，点击下图所示，选择显示所有设备，根据你的设备选择相应的磁盘在弹出的窗口中输入：名称：Macintosh HD；格式：Mac OS扩展（日志式）；方案：GUID分区图，点击抹掉；

    然后等待操作结束，点击完成，通过菜单选择退出磁盘工具或者按窗口左上角红色按钮离开磁盘工具

    返回到安装界面，选择安装macOS，点击继续 一直同意到安装完成 安装完成后先打开终端，输入几行命令： sudo spctl --master-disable sudo kextcache -i /

    # 五、进入系统能正常使用后需要把需要把U盘上的clover引导系统的工具及其引导用的EFI文件拷贝到本机硬盘的EFI分区

    打开终端 输入diskutil list 先挂载磁盘EFI分区 sudo diskutil mount disk0s1 open . 可打开finder，在finder里进行标签以防止混乱

    然后再挂载U盘EFI分区 sudo diskutil mount disk2s1 在finder里面左侧会显示挂载了两个EFI分区，将U盘EFI目录全部拷贝到磁盘的EFI分区即可
